Projeto de densenvolvimento pessoal para demonstrar a integração do frontend com o backend.

📋 Descrição
Projeto simples que implementa uma aplicação fullsatck, com api desenvolvida em Python e Django Rest Framework e o frontend desenvolvido com React.js e Material UI.

Tecnologias Utilizadas

    Django
    Django Rest Framework
    React.js
    Material UI
    Vercel

🔧 Requisitos

    Python 3.7+
    pip

📦 Instalação

Siga os passos abaixo para rodar o projeto localmente:

    Clone o repositório:

    git clone https://github.com/jandsonrj/pokemon-ionic-app.git

    Acesse o diretório do projeto:

    cd KanbanTaskBoard

    Instale as dependências:

    npm install

    Inicie a aplicação:
      Backend:  python .\manage.py runserver
      Fontend: npm run dev
    

A aplicação será automaticamente aberta no seu navegador na porta: http://localhost:8100/
